### Question 1
For a given dataset, which software (and version) was used in the activity that generated it?

```sparql
SELECT ?dataset ?datasetTitle ?activity ?software ?softwareTitle ?version
WHERE {
  ?dataset a dcat:Dataset ;
           prov:wasGeneratedBy ?activity .

  ?activity rdip:usedSoftware ?software .

  OPTIONAL { ?dataset  rdip:title   ?datasetTitle . }
  OPTIONAL { ?software rdip:title   ?softwareTitle . }
  OPTIONAL { ?software rdip:version ?version . }
}
ORDER BY ?dataset ?software
```

**Expected results (v2 examples):** 3 datasets × their respective software tools.
- AnnotatedTigerRhinoDataset → YOLOv8 8.0.1, Roboflow web-2025-01
- AnonymizedTranscriptsDataset → NVivo 14.23.0
- ThaiRetinaDataset → OpenCV 4.9.0

### Question 2
Which formal methods or protocols were employed in a specific data production activity?

```sparql
SELECT ?activity ?activityType ?activityTitle ?method ?methodTitle ?methodDesc ?methodDoi
WHERE {
  ?activity a ?activityType ;
            rdip:usedMethod ?method .

  # Restrict to production/analysis activities only
  VALUES ?activityType {
    rdip:DataProductionActivity
    rdip:DataAnalysisActivity
    rdip:DataCollectionActivity
    rdip:DataProcessingActivity
  }

  OPTIONAL { ?activity rdip:title       ?activityTitle . }
  OPTIONAL { ?method   rdip:title       ?methodTitle . }
  OPTIONAL { ?method   rdip:description ?methodDesc  . }
  OPTIONAL { ?method   rdip:methodDoi   ?methodDoi   . }
}
ORDER BY ?activity ?method
```

**Expected results:** 3 activity–method pairs, each with a methodDoi where populated.

### Question 3
For a given publication, which project produced it and who was the Principal Investigator (PI)?

```sparql
SELECT DISTINCT
  ?article ?articleTitle
  ?project ?projectTitle
  ?pi ?piName ?orcid
WHERE {
  ?project rdip:hasOutput ?article .
  ?article a bibo:Article .

  OPTIONAL { ?article rdip:title ?articleTitle . }
  OPTIONAL { ?project rdip:title ?projectTitle . }

  ?project  rdip:hasActivity ?activity .
  ?activity rdip:hasActivityRole ?role .

  ?role a rdip:RoleInActivity ;
        rdip:roleLabel       ?roleLabel ;
        rdip:rolePerformedBy ?pi .

  FILTER (REGEX(?roleLabel, "Principal Investigator", "i"))

  OPTIONAL { ?pi rdfs:label   ?piName . }
  OPTIONAL { ?pi rdip:orcidId ?orcid  . }
}
ORDER BY ?article ?pi
```

**Expected results:** 3 rows — one PI per project/article pair.

### Question 4
For a given project, which datasets were produced and what are their access levels and landing pages?

```sparql
SELECT ?project ?projectTitle ?dataset ?datasetTitle ?accessLevel ?landingPage ?dataLicense
WHERE {
  ?project a rdip:ResearchProject ;
           rdip:hasOutput ?dataset .
  ?dataset a dcat:Dataset .

  OPTIONAL { ?project rdip:title       ?projectTitle . }
  OPTIONAL { ?dataset rdip:title       ?datasetTitle . }
  OPTIONAL { ?dataset rdip:accessLevel ?accessLevel . }
  OPTIONAL { ?dataset rdip:landingPage ?landingPage . }   # property name kept as in v2 [F2]
  OPTIONAL { ?dataset rdip:dataLicense ?dataLicense . }  # NEW v2 property
}
ORDER BY ?project ?dataset
```

**Expected results:** 3 datasets, each with accessLevel, landingPage (where set), and dataLicense.

### Question 5
Given a specific output (dataset, software or publication), what other outputs were produced by the same project?

```sparql
SELECT DISTINCT ?givenOutput ?project ?projectTitle ?otherOutput ?otherType ?otherTitle
WHERE {
  ?givenOutput rdip:isOutputOf ?project .
  ?project     rdip:hasOutput  ?otherOutput .

  FILTER (?otherOutput != ?givenOutput)

  OPTIONAL { ?project    rdip:title ?projectTitle . }
  OPTIONAL { ?otherOutput a ?otherType . }
  OPTIONAL { ?otherOutput rdip:title ?otherTitle . }
}
ORDER BY ?givenOutput ?otherOutput
```

**Expected results:**
- ChitwanAI: dataset ↔ article
- NepalStartups: dataset ↔ article
- ThaiMedAI: dataset ↔ software ↔ article (6 pairs)

### Question 6
What was the specific role of a person in particular research activities and projects?

```sparql
SELECT ?person ?personName ?activity ?activityType ?activityTitle ?project ?projectTitle ?roleLabel
WHERE {
  ?activity a ?activityType ;
            rdip:hasActivityRole ?role ;
            rdip:isPartOfProject ?project .

  # Broaden from DataProductionActivity to all activity types [updated scope]
  FILTER (?activityType IN (
    rdip:DataProductionActivity, rdip:DataAnalysisActivity,
    rdip:DataCollectionActivity, rdip:DataProcessingActivity,
    rdip:SoftwareDevelopmentActivity
  ))

  ?role rdip:rolePerformedBy ?person ;
        rdip:roleLabel       ?roleLabel .

  OPTIONAL { ?person   rdfs:label ?personName . }
  OPTIONAL { ?activity rdip:title ?activityTitle . }
  OPTIONAL { ?project  rdip:title ?projectTitle . }
}
ORDER BY ?person ?project ?activity
```

### Question 7
For a specific dataset, which activity generated it, using what software, under which project, and who was the agent responsible?

```sparql
SELECT ?datasetTitle ?activityTitle ?softwareName ?softwareVersion ?projectName ?agentName ?roleName
WHERE {
  ?dataset a dcat:Dataset ;
           rdip:title ?datasetTitle ;
           prov:wasGeneratedBy ?activity .

  ?activity rdip:title ?activityTitle ;
            rdip:usedSoftware ?software ;
            rdip:isPartOfProject ?project .

  ?software rdip:title ?softwareName .
  OPTIONAL { ?software rdip:version ?softwareVersion . }

  ?project rdip:title ?projectName .

  ?activity rdip:hasActivityRole ?role .
  ?role     rdip:rolePerformedBy ?agent ;
            rdip:roleLabel       ?roleName .
  ?agent    rdfs:label ?agentName .
}
ORDER BY ?projectName ?datasetTitle ?agentName
```

**Note on row multiplicity:** This query produces one row per (dataset × software × agent × role) combination.
ThaiMedAI will produce multiple rows per dataset because there are 3 agents and 1 software tool on the processing activity.
Use `SELECT DISTINCT` and restrict with `FILTER` if a flatter result is needed.

### Question 8 *(NEW — v2 reproducibility)*
For a given computational activity, what were the exact hyperparameters and random seed used?

```sparql
SELECT ?activity ?activityTitle ?paramName ?paramValue ?paramType ?isRandomSeed
WHERE {
  ?activity rdip:hasParameter ?param .

  OPTIONAL { ?activity  rdip:title           ?activityTitle . }
  OPTIONAL { ?param     rdip:parameterName   ?paramName . }
  OPTIONAL { ?param     rdip:parameterValue  ?paramValue . }
  OPTIONAL { ?param     rdip:parameterDataType ?paramType . }

  BIND(EXISTS { ?param a rdip:RandomSeed } AS ?isRandomSeed)
}
ORDER BY ?activity DESC(?isRandomSeed) ?paramName
```

**Expected results:** 4 params for ChitwanAI (LR, batch, epochs, seed), 5 for ThaiMedAI (LR, batch, epochs, dropout, seed).

### Question 9 *(NEW — v2 reproducibility)*
What computing environment (OS, GPU, CUDA, Docker image) was used for a given activity?

```sparql
SELECT ?activity ?activityTitle ?osVersion ?gpuModel ?cudaVersion ?ramGB ?specType ?specUri ?imageDigest
WHERE {
  ?activity rdip:executedIn ?env .

  OPTIONAL { ?activity rdip:title              ?activityTitle . }
  OPTIONAL { ?env      rdip:osVersion          ?osVersion . }
  OPTIONAL { ?env      rdip:gpuModel           ?gpuModel . }
  OPTIONAL { ?env      rdip:cudaVersion        ?cudaVersion . }
  OPTIONAL { ?env      rdip:ramGB              ?ramGB . }
  OPTIONAL {
    ?env rdip:hasEnvironmentSpec ?spec .
    OPTIONAL { ?spec rdip:specType    ?specType . }
    OPTIONAL { ?spec rdip:specUri     ?specUri . }
    OPTIONAL { ?spec rdip:imageDigest ?imageDigest . }
  }
}
ORDER BY ?activity
```

### Question 10 *(NEW — v2 reproducibility)*
What are the evaluation results for a given project, and on which dataset split were they measured?

```sparql
SELECT ?project ?projectTitle ?activity ?metricName ?metricValue ?splitLabel ?evalDate
WHERE {
  ?project  rdip:hasActivity  ?activity .
  ?activity rdip:generatesResult ?result .

  OPTIONAL { ?project  rdip:title          ?projectTitle . }
  OPTIONAL { ?result   rdip:metricName     ?metricName . }
  OPTIONAL { ?result   rdip:metricValue    ?metricValue . }
  OPTIONAL { ?result   rdip:splitLabel     ?splitLabel . }
  OPTIONAL { ?result   rdip:evaluationDate ?evalDate . }
}
ORDER BY ?project ?metricName
```

**Expected results:** ChitwanAI → mAP@0.5=0.873, precision=0.891; ThaiMedAI → AUC=0.961, sensitivity=0.923, specificity=0.947; NepalStartups → identified_themes=6.

### Question 11 *(NEW — v2 provenance)*
Trace the full lineage of a dataset back to its raw source through derivation chains.

```sparql
SELECT ?project ?projectTitle ?activity ?metricName ?metricValue ?splitLabel ?evalDate
WHERE {
  # Constrain types explicitly to block reasoner noise
  ?project  a rdip:ResearchProject .
  ?activity a ?actType .
  FILTER ( ?actType = rdip:DataAnalysisActivity 
        || ?actType = rdip:DataProductionActivity )

  ?project  rdip:hasActivity    ?activity .
  ?activity rdip:generatesResult ?result .
  ?result   a rdip:EvaluationResult .   # constrain result type too

  OPTIONAL { ?project  rdip:title          ?projectTitle . }
  OPTIONAL { ?result   rdip:metricName     ?metricName . }
  OPTIONAL { ?result   rdip:metricValue    ?metricValue . }
  OPTIONAL { ?result   rdip:splitLabel     ?splitLabel . }
  OPTIONAL { ?result   rdip:evaluationDate ?evalDate . }
}
ORDER BY ?project ?metricName
```

**Expected results:**
- AnnotatedTigerRhinoDataset ← RawCameraTrapImages
- AnonymizedTranscriptsDataset ← RawInterviewTranscripts
- ThaiRetinaDataset ← RawRetinalImages
